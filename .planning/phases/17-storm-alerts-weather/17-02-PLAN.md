         id: string;

    2. Create `src/lib/storm-alerts/queries.ts`:
       - getActiveStormAlertsForOrg(orgId: string): Promise<StormAlert[]>
         - Join organization_profiles to get hqLat, hqLng, serviceRadiusMiles
         - Query leads where sourceType = 'storm' AND deadlineDate > now()
         - Use haversine SQL formula to filter by distance: `3959 * acos(cos(radians(hqLat)) * cos(radians(lead.lat)) * cos(radians(lead.lng) - radians(hqLng)) + sin(radians(hqLat)) * sin(radians(lead.lat))) <= serviceRadiusMiles`
         - Order by deadlineDate ASC (soonest expiring first)
         - Limit 10 (don't flood the banner)
         - Map to StormAlert type: { id, title, description, severity, city, state, lat, lng, expiresAt: deadlineDate, sourceUrl }

       - getRoofingSubscribersInStormArea(stormLat: number, stormLng: number): Promise<SubscriberInfo[]>
         - Query organizations WHERE industry = 'roofing'
         - Join organization_profiles for hqLat, hqLng, serviceRadiusMiles
         - Filter: haversine distance from stormLat/stormLng to hqLat/hqLng <= serviceRadiusMiles
         - Join member + user tables to get email addresses
         - Return: { orgId, orgName, userId, userName, email, hqLat, hqLng, serviceRadiusMiles }[]
