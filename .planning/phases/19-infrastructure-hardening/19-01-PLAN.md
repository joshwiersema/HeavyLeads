Rewrite `src/lib/geocoding.ts` to implement a 3-layer geocoding strategy:

1. **Cache lookup**: Hash the normalized address, query `geocoding_cache` for a non-expired row. If found and `lat` is not null, return cached result. If found and `lat` is null, return null coords (known-bad address).

2. **Google Maps primary**: If cache miss, call Google Maps Geocoding API (existing logic). On success, insert result into cache with 90-day expiry. On `OVER_QUERY_LIMIT` or `REQUEST_DENIED` status from Google API response body (`data.status === "OVER_QUERY_LIMIT"`), fall through to Nominatim.

3. **Nominatim fallback**: If Google fails or is quota-exceeded, call Nominatim. Rate limit to 1 request/second. Insert result into cache with 90-day expiry.

Implementation details:

```typescript
import { createHash } from "crypto";
import { db } from "@/lib/db";
import { geocodingCache } from "@/lib/db/schema/geocoding-cache";
import { eq, gt } from "drizzle-orm";

interface GeocodingResult {
  lat: number | null;
  lng: number | null;
  formattedAddress: string;
}

/** Normalize address for consistent hashing */
function normalizeAddress(address: string): string {
  return address.toLowerCase().trim().replace(/\s+/g, " ");
}

/** SHA-256 hash of normalized address */
function hashAddress(address: string): string {
  return createHash("sha256").update(normalizeAddress(address)).digest("hex");
}

let lastNominatimCall = 0;

async function nominatimGeocode(address: string): Promise<GeocodingResult> {
  // Enforce 1 req/sec rate limit
  const now = Date.now();
  const elapsed = now - lastNominatimCall;
  if (elapsed < 1000) {
    await new Promise((resolve) => setTimeout(resolve, 1000 - elapsed));
  }
  lastNominatimCall = Date.now();

  const url = `https://nominatim.openstreetmap.org/search?q=${encodeURIComponent(address)}&format=json&limit=1`;
  const response = await fetch(url, {
    headers: {
      "User-Agent": "GroundPulse/1.0 (construction-lead-intelligence)",
    },
  });

  if (!response.ok) {
    throw new Error(`Nominatim API error: ${response.status} ${response.statusText}`);
  }

  const results = await response.json();
  if (!results || results.length === 0) {
    return { lat: null, lng: null, formattedAddress: address };
  }

  return {
    lat: parseFloat(results[0].lat),
    lng: parseFloat(results[0].lon),
    formattedAddress: results[0].display_name || address,
  };
}
```

The main `geocodeAddress` function flow:
1. Compute `addressHash = hashAddress(address)`
2. Query `SELECT * FROM geocoding_cache WHERE address_hash = $1 AND expires_at > NOW()` using Drizzle
3. If cache hit: return `{ lat, lng, formattedAddress }` from cached row
4. If cache miss: try Google Maps API
   - If Google returns `OVER_QUERY_LIMIT` or `REQUEST_DENIED`: try Nominatim
   - If Google returns valid result: cache it and return
   - If Google has no API key set: try Nominatim directly
5. If Nominatim used: cache the result (provider: "nominatim") and return
6. On any geocoding failure: cache null coords to prevent retrying, return `{ lat: null, lng: null, formattedAddress: address }`

Cache insert uses `onConflictDoUpdate` on `addressHash` to handle races.

The function signature stays identical: `geocodeAddress(address: string): Promise<GeocodingResult>` -- no breaking changes for `pipeline.ts` consumers.
