Auth & Billing

Identity & payments
BA
Better Auth 1.5.5
Full-featured auth framework. Email/password login, organization plugin (multi-tenant, 50 member limit), Drizzle adapter, and cookie-based sessions.
S
Stripe 20.4.1
Subscription billing with 7-day free trial via Stripe Checkout. Org-level customers, webhook events, conditional setup fee logic. Client SDK for checkout.

UI & Styling

Design system
Tw
Tailwind CSS 4.x
Utility-first CSS framework. PostCSS integration, custom theme variables in globals.css, and tw-animate-css for animations.
sh
shadcn/ui 4.0.6
Copy-paste component library. Badge, Button, Card, Checkbox, Input, Label, Select, Skeleton, Slider, and Sonner (toasts).
Lu
Lucide React 0.577
Icon library. Clean, consistent SVG icons used throughout the app for navigation, status indicators, and actions.
cva
CVA + clsx + tailwind-merge
Component variant system (class-variance-authority), class composition (clsx), and intelligent Tailwind class merging.
