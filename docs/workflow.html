Background: Daily Automation

Vercel Cron — 6:00 AM UTC

Runs automatically every day with no user action required.

Cron Job Fires

Vercel triggers GET /api/cron/scrape at 6:00 AM UTC daily. Authenticated via CRON_SECRET Bearer token.

Pipeline Scrapes All Sources

Initializes all registered adapters (permits, bids, news, deep-web) and runs them sequentially. Each adapter scrapes its data source and stores new leads in the database.

Results Recorded

Pipeline run is tracked in the pipeline_runs table with: records scraped, records stored, duration, and status.

Email Digest Sent

After scraping completes, an email digest of new leads is generated and sent to users with saved searches (if Resend is configured).

Result: Every morning, sales reps open HeavyLeads and see fresh leads they would have otherwise missed.
